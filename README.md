# Chạy và triển khai ứng dụng AI Studio của bạn

Tài liệu này chứa mọi thứ bạn cần để chạy ứng dụng cục bộ.

Xem ứng dụng của bạn trong AI Studio: https://ai.studio/apps/bundled/ask_the_manual

## Chạy cục bộ

**Yêu cầu:** Node.js

1. Cài đặt các phụ thuộc:
   `npm install`
2. Thiết lập `GEMINI_API_KEY` trong [.env.local](.env.local) thành khóa API Gemini của bạn
3. Chạy ứng dụng:
   `npm run dev`
